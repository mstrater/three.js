[page:Mesh] →

[name]

A mesh that always faces a specified point (usually a camera). Optionally, the mesh can be constrained to a specified axis. If so, it will face the specified point as much as possible while remaining aligned with the axis. If not constrained, the axis can be used as a custom up vector for the mesh. Requires a call to the update method after each change.

Constructor

[name]([page:Geometry geometry], [page:Material material], [page:Vector3 point], [page:Vector3 axis], [page:Boolean constrainedToAxis])

geometry — Geometry for the mesh.
material — Material for the mesh.
point — The point the mesh should face.
axis — The axis from the geometry origin that the mesh should be constrained to. If not constrained, defines a custom up vector for the mesh. Default is (0,1,0).
constrainedToAxis — Boolean for whether the mesh should be constrained to the specified axis. Default is false.
Creates a [name] facing the specified point and optionally constrained to the specified axis. Requires a call to update before render.

Properties

[property:Vector3 point]

Gets or sets the point the mesh faces. Requires an update if setting to a new value.

[property:Vector3 axis]

Gets or sets the axis that the mesh is constrained to. If not constrained, sets the mesh up vector. Requires an update if setting to a new value.

[property:Boolean constrained]

Gets or sets the whether the mesh is constrained to axis. Requires an update if setting to a new value.

Methods

[method:null update]()

Updates the rotation of the mesh to look toward the specified point.
